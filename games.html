If you're looking to play OneShot, you've come to the right place. Each of these have descriptions that tell you what the version does, so make sure you get the right one for your console.

Starting with this game! To experience the full game of OneShot you have to pay for one of the main versions of the game, but who says you can't have a little fun now?

This version of the game allows you to play a small version of the game, to get a feel for what it'll play like before you buy it!

Outside of the demo, we now have our main games for OneShot, including...

OneShot

The base version of the game. This version gives you the original intended experience, where you have to save the world with Niko.

This version works for PC play, but since it needs access to the files, it doesn't work for other consoles.

Buy it here:

OneShot: World Machine Edition

This is the base version of the game, made for devices without an easy to access file directory. Otherwise it's very similar to the base version, but includes many new customizable features and artwork.

This version works for most consoles.

Buy it here:

OneShot: The Pancake Episode

This is a side game for the 10th anniversary of OneShot. It's a lot shorter than the base game, but it's meant to be a fun side story to celebrate the game.

This version doesn't access the files at all, and so it's technically compatible with most platforms, but officially it's only available for PC.

Buy it here:

OneShot: Original

The original version of the game. This is sort of like a rough version of OneShot, where some of the characters may have acted different, and the consequences may have been far more drastic.

This version works for PC play only, but it is no longer publicly available through regular means.

You only have one shot.
